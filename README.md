#  Contact Book App

A simple command-line Contact Book built in Python. Save, view, search, and delete contacts — no database, just plain file storage.

## Features
-  Add a contact (name, phone, email)
-  View all saved contacts
-  Search contacts by name
-  Delete a contact
-  Data persists in `contacts.txt`, no setup required

## Tech Stack
Python 3 — standard library only, no external packages.

## Getting Started

git clone https://github.com/your-username/contact-book-app.git
cd contact-book-app
python contact_book.py
```

## Usage


===== CONTACT BOOK =====
1. Add Contact
2. View All Contacts
3. Search Contact
4. Delete Contact
5. Exit
```

Pick an option and follow the prompts. Contacts are saved automatically in `contacts.txt`.
